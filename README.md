TO DO BUDDY
Overview

TO DO BUDDY is a modern, user-friendly task management web application designed to help users efficiently organize, track, and manage their daily tasks. The application provides a clean dashboard layout with real-time task updates and persistent storage, ensuring a smooth and reliable user experience.

The application is built using core web technologies and follows a simple yet effective design approach commonly used in real-world productivity tools.

Features
Task Management

Add new tasks with title, category, priority, and deadline

Automatically records task creation time

Edit, complete, and delete tasks

Visual indication for completed and overdue tasks

Organization and Filtering

Categorize tasks into Work, Personal, and Study

Assign priority levels (High, Medium, Low)

Filter tasks by status (All, Pending, Completed)

Filter tasks by category and priority

Search tasks instantly by title

User Interface

Split-screen dashboard layout

Left panel for task creation

Right panel for task listing and filtering

Clean, card-based task display

Dark mode toggle

Responsive and modern UI design

Data Persistence

Tasks are stored using browser LocalStorage

Data remains available after page refresh or browser restart

Technology Stack

HTML5 for structure

CSS3 for styling and layout

JavaScript (ES6) for functionality and logic

Browser LocalStorage for persistent data storage

Application Layout

Top Header: Application branding (TO DO BUDDY)

Left Panel: Task creation form

Right Panel: Task list, filters, search, and counters

How It Works

Users enter task details in the form panel

The application automatically records the creation timestamp

Tasks are displayed dynamically in the task panel

Filters and search update the task list in real time

All task data is saved in LocalStorage for persistence

Installation and Usage

Download or clone the project files

Open the index.html file in any modern web browser

Start adding and managing tasks immediately

No additional setup or dependencies are required.

Future Enhancements

Drag and drop task reordering

Task editing using modal dialogs

Notification reminders for upcoming deadlines

User authentication and backend integration

Cloud-based data storage

Project Purpose

This project demonstrates practical usage of JavaScript DOM manipulation, state management, filtering logic, and modern UI design principles. It is suitable for academic submission, portfolio projects, and frontend development practice.
